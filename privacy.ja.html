Stickee プライバシーポリシー

施行日: 2026-06-01

Stickee（以下「本サービス」）は、利用者のプライバシーを尊重します。本ポリシーでは、当社が収集する情報、利用方法、保管および廃棄の方法について説明します。

1. 収集する個人情報

  • Apple Sign-In 情報: Apple ユーザー識別子、メールアドレス（Apple Private Relay 対応）
  • プロフィール: ニックネーム、プロフィール画像（任意）
  • ユーザーコンテンツ: テキスト投稿（最大200文字）、添付画像、1:1 ダイレクトメッセージ、返信（チェーンノート）
  • 位置情報: 投稿作成時点の緯度・経度（高精度）
  • 自動収集: セッショントークン、IPアドレス、デバイス情報（アプリバージョン、OSバージョン）

以下は収集しません:

  • 広告識別子（IDFA）
  • 連絡先、カレンダー、ヘルスデータ、決済情報

2. 利用目的

  • Apple Sign-In によるアカウント認証
  • 近くにいる他のユーザーの投稿表示（位置情報による距離計算）
  • 1:1 ダイレクトメッセージの送受信
  • 不適切なコンテンツの通報処理およびモデレーション
  • サービスの安定性（エラー追跡、不正利用防止）
  • 同じエリア（半径数キロメートル以内）にいる他のユーザーのクライアントに投稿の位置座標が共有され、各クライアントで距離計算とキャンバス配置に使用されます。位置座標自体は UI に表示されません。

3. 保有および利用期間

項目 期間
アカウント情報 アカウント削除と同時に永久に削除
投稿（ふせん） 作成から6時間後にサーバーから自動削除（ユーザーは自分で先に削除可能）
ダイレクトメッセージ アカウント存続期間中
通報データ 90日間（モデレーション検討目的）

法令上の保有義務がある項目はありません。

4. 第三者提供および処理委託

  • Apple Inc. — Apple Sign-In 認証処理（目的限定）
  • Supabase Inc. — データベース／Realtime／Storage／認証サービス（米国リージョン；正確なリージョンは運営者の確認後、本文書を更新）

広告業者やマーケティング・分析サービスにはデータを提供しません（v1.0 時点）。変更がある場合は事前に告知します。

5. 利用者の権利

  • 閲覧: プロフィール画面でニックネーム／メールアドレスを確認
  • 修正: プロフィール編集（ニックネーム変更）
  • 削除: プロフィール → アカウント削除（即時に永久削除、復旧不可）
  • 同意の撤回: アカウント削除をもって代える
  • 処理停止: アカウント削除またはサインアウト

6. 14歳未満の児童

本サービスは17歳以上の利用を推奨しています（App Store 年齢制限17+）。14歳未満であることが確認されたアカウントは直ちに削除します。

7. 安全管理措置および廃棄方法

安全管理:

  • 伝送区間: HTTPS（Supabase エンドポイント）
  • 保管区間: Supabase サーバー側暗号化（AES-256）
  • アクセス制御: Supabase RLS（Row Level Security）によるユーザー単位の分離
  • 保管場所: Supabase 米国リージョン

廃棄方法:

  • アカウント情報・コンテンツ・DM・通報データ: アカウント削除または保有期間経過時にデータベースレコードを即時に永久削除（SQL DELETE）し、復元しません。ユーザーIDは外部キー CASCADE により関連するすべてのテーブルから同時に削除されます。
  • 画像ファイル: Supabase Storage の削除 API により永久削除します。
  • バックアップ: 通常のバックアップサイクルに沿って、バックアップからも順次削除されます。
